## Content Guidelines

### Writing Sermon Titles
✅ DO: "Walking in Divine Purpose"
❌ DON'T: "sunday sermon nov 24"

**Tips:**
- Use title case
- Be descriptive
- Keep it under 60 characters
- Make it meaningful to visitors

### Writing Descriptions
✅ DO: "Understanding and fulfilling God's unique plan for your life through faith and obedience."
❌ DON'T: "Good sermon about purpose"

**Tips:**
- Write 1-2 complete sentences
- Explain what people will learn
- Use proper grammar and punctuation
- Keep it under 160 characters

### Dates
Always use format: `YYYY-MM-DD` (2024-11-24)

### Images

**Best Practices:**
- Use high-quality photos
- Landscape orientation preferred (4:3 or 16:9)
- Keep file size under 200KB
- Use meaningful file names
- Include people's faces when possible

**Image Sizes:**
- Sermon/Event cards: 800x600px
- Hero backgrounds: 1920x1080px
- Ministry photos: 600x600px

**Current times shown:**
- Wednesday Bible Study: 18:00-19:30
- Saturday Prayer Meeting: 12:00-13:00
- Sunday Service: 11:00-13:00

**To update:**
1. Note the new times
2. Email web administrator with changes
3. Specify which service(s) changed

### Adding a New Ministry

**Information needed:**
- Ministry name
- Description (2-3 sentences)
- Leader name (optional)
- Photo (optional)

**Example:**
```
Ministry: Young Adults Fellowship
Description: A dynamic community for young adults aged 18-35 to grow in faith, build friendships, and discover their purpose in Christ.
Leader: Brother Samuel Johnson
Photo: young-adults.jpg
```

### Sharing a Testimony

**Information needed:**
- Person's name (or "Anonymous")
- Testimony title
- Brief description (100-150 words)
- Date of testimony
- Category (Healing, Financial, Salvation, etc.)

**Example:**
```
Name: Sister Grace Thompson
Title: Divine Healing from Chronic Pain
Description: After suffering from chronic back pain for 5 years, I experienced complete healing during our prayer meeting. Doctors had said I would need surgery, but God had a different plan...
Date: November 15, 2024
Category: Healing
```

## Image Management

### Adding Images

1. **Prepare Your Image**
   - Take a high-quality photo
   - Crop to appropriate size
   - Reduce file size if needed (use: tinypng.com)

2. **Send to Administrator**
   - Email the image
   - Specify where it should appear
   - Include caption/description

### Image Requirements

| Type | Size | Format |
|------|------|--------|
| Sermon thumbnail | 800x600px | JPG |
| Event photo | 800x600px | JPG |
| Ministry photo | 600x600px | JPG |
| Hero background | 1920x1080px | JPG |
| Church logo | 512x512px | PNG |

### Getting Good Photos

**Do:**
- Use good lighting
- Include people
- Show activity/engagement
- Keep backgrounds clean
- Use landscape orientation

**Don't:**
- Use blurry photos
- Include copyrighted material
- Use photos without permission
- Upload huge file sizes

## Frequently Asked Questions

### Q: How often should we update content?

**Recommended:**
- Sermons: Weekly (after Sunday service)
- Events: As they're scheduled
- Testimonies: Monthly or as received
- General content: Quarterly review

### Q: Who can make updates?

Only people with admin credentials. Contact church leadership to request access.

### Q: How long does it take for changes to appear?

- Via admin dashboard: Immediately
- Via developer: 1-2 business days

### Q: Can I preview changes before publishing?

Yes, once the admin system is fully implemented. For now, request a preview from the developer.

### Q: What if I make a mistake?

Don't worry! Contact the administrator immediately to correct it. Most changes can be reversed.

### Q: Can I update the website from my phone?

Yes! The website is mobile-friendly, including the admin area.
