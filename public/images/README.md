- church-exterior.jpg - Exterior photo of the church building
- service-1.jpg - Sunday service photo

## Recommended Image Sizes

- Hero backgrounds: 1920x1080px or larger
- Card images: 800x600px
- Thumbnails: 400x300px
- Logo: 512x512px (transparent PNG)

## Image Optimization

All images in the public directory are automatically optimized by Next.js Image component. Use WebP or AVIF formats when possible for better compression.
